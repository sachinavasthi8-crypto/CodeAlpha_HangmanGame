🎮 Hangman Game — Python

A simple text-based Hangman game built with Python.

📌 About the Project

In this project, the player has to guess a hidden word one letter at a time.
The game ends when the player either guesses the word correctly or runs out of 6 chances.
An ASCII art hangman is displayed after every wrong guess.

✨ Features

5 predefined random words
ASCII art hangman that updates on every wrong guess
Tracks guessed letters
Play again option at the end
Simple console-based input/output

🛠️ Technologies Used

Python 3
Libraries: random (built-in)

▶️ How to Run

Make sure Python is installed on your system
Download or clone this repository
Open terminal and run:

python task1_hangman.py

🧠 Key Concepts Used

random module
while loop
if-else conditions
Strings and Lists
